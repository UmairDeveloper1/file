HOW TO CREATE THE BUILDING

1- New Record

THis button will allow users to create a new master record in the Building Master- the user will feed the data in the relative fields; some fields are text fields such as building Name(English)and some are the list of values (LOV) that allow the user to choose one value from a list .

2- TAB

The building master contains the tab - the tab itself provides the navigation to its child screen i.e. Building specification that allow users to enter particular building's specification and test of details.

3- Link Building with Owner

Upon creation of the new Building - user is required to enter the landlord (owner of the building). When linking building with Owner - the Landlord name and contact # will fetch automatically from Landlord master.
